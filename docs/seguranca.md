# Segurança, Privacidade e Curadoria

Este repositório publica uma versão curada das skills.

## O que foi excluído

- `.DS_Store`;
- `__pycache__`;
- arquivos `.pyc`;
- skills de sistema em `.system`;
- caches locais;
- caminhos absolutos privados em arquivos publicados.

## O que revisar antes de usar profissionalmente

- dados de clientes;
- números de processo;
- documentos pessoais;
- documentos médicos ou financeiros;
- valores sensíveis;
- sigilo profissional;
- autorização do cliente;
- política da ferramenta de IA utilizada.

## Petição Completa

A skill `peticao-completa` inclui modelo DOCX, scripts e referências. Antes de usar no seu escritório, revise:

- logotipo;
- assinatura;
- OAB;
- rodapé;
- endereço;
- e-mail;
- estilo de redação;
- regras de conferência jurisprudencial;
- fontes e dependências locais.

O arquivo `references/sintese-cerebro.md` foi publicado como orientação genérica. Cada usuário deve gerar sua própria síntese a partir do seu acervo, já anonimizado.

## Instalação de terceiros

Antes de instalar qualquer skill de outro repositório, revise origem, arquivos, scripts e permissões. Não execute comandos de repositórios desconhecidos sem auditoria mínima.
