# Design

## Direção

Landing page editorial e funcional para advocacia brasileira. O visual deve transmitir método, confiança e material de trabalho, sem parecer painel SaaS genérico.

## Paleta

- Verde jurídico escuro para autoridade e ação.
- Papel quente para leitura longa.
- Cobre discreto para marcação e hierarquia.
- Azul pontual para foco e acessibilidade.

## Tipografia

- Display: Bricolage Grotesque.
- Texto: Alegreya Sans.
- Mono: fonte do sistema para comandos.

## Interface

- Primeira dobra com comando de instalação visível.
- Catálogo com busca e filtros.
- Cards expansíveis para reduzir ruído.
- Botões de cópia para comandos.
- Links de pasta e download individual.

## Restrições

- Sem estética de marketing agressivo.
- Sem promessa de substituição do advogado.
- Sem blocos de texto longos demais na primeira dobra.
- Comandos devem quebrar linha corretamente no mobile.
